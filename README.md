# Kiddo

This is a todo list application that can restrict internet access and other privileges if kids don't complete their assigned todos.

So nag them until they do what we told them needs to be done :raised_eyebrow:

## Features that may be implemented
- [ ] Shut off internet in the FRITZ!Box router if important todos are open
- [ ] Shut down smart power outlets if todos are open
- [ ] Set new PIN in online streaming services
